# PYTHON TUPLE

A tuple is used to store multiple items in one variable.

## What is a tuple?
A tuple is a collection of items.
It is written using round brackets `( )`.

Example:
fruits = ("apple", "banana", "mango")

## Tuple features
- Ordered
- Unchangeable
- Allows duplicate values
- Can store different data types

## Ordered
Tuple items stay in the same order.

Example:
fruits = ("apple", "banana", "mango")

## Unchangeable
Once a tuple is created, you cannot change, add, or remove items.

Example:
fruits[1] = "orange"   # This will give an error

## Allows duplicates
A tuple can contain the same value more than once.

Example:
numbers = (1, 2, 2, 3)

## Can store different data types
A tuple can have numbers, text, and True/False values together.

Example:
data = (10, "hello", True, 5.5)

## Indexing
Tuple items have indexes.
The first item is at index 0.

Example:
fruits[0] gives the first item

## Single item tuple
If a tuple has only one item, add a comma after it.

Example:
one_item = ("apple",)

## Why use tuples?
Use tuples when the data should not be changed.
Examples:
- days of the week
- coordinates
- fixed values
- record data

## Important points
- Tuples use round brackets `( )`
- They are ordered
- They are immutable
- They allow duplicates
- They can store different types
