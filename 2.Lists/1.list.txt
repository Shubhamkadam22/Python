PYTHON LISTS

--properties of lists 















A list is used to store multiple items in one variable.

WHAT IS A LIST?
-
A list is a collection of items.
It is written using square brackets [ ].

Example:
fruits = ["apple", "banana", "mango"]

LIST FEATURES

- Ordered
- Changeable
- Allows duplicate values
- Can store different data types

Example:
items = [10, "hello", True]

INDEXING

List items have positions called indexes.
The first item is at index 0.

Example:
fruits[0] gives apple
fruits[1] gives banana

CHANGING ITEMS

Lists are changeable, so you can update values.

Example:
fruits[1] = "orange"

ADDING ITEMS

append() adds one item at the end.

Example:
fruits.append("grapes")

REMOVING ITEMS

remove() deletes a specific item.

Example:
fruits.remove("apple")

LENGTH OF LIST

Use len() to find how many items are in a list.

Example:
len(fruits)

WHY USE LISTS?

Use lists when you want to store many related values together.
Examples:
- names of students
- fruits in a basket
- marks of students
- shopping items

IMPORTANT POINTS

- Index starts from 0
- Lists use [ ]
- Items can be changed
- Lists can grow and shrink
