# PYTHON LIST

A list is used to store multiple items in one variable.

## What is a list?
A list is a collection of items.
It is written using square brackets `[ ]`.

Example:
fruits = ["apple", "banana", "mango"]

## List features
- Ordered
- Changeable
- Allows duplicate values
- Can store different data types

## Ordered
List items stay in the same order.

Example:
fruits = ["apple", "banana", "mango"]

## Changeable
Lists can be changed after creation.
You can update, add, or remove items.

Example:
fruits[1] = "orange"

## Allows duplicates
A list can contain the same value more than once.

Example:
numbers = [1, 2, 2, 3]

## Can store different data types
A single list can contain numbers, text, and True/False values.

Example:
mixed = [10, "hello", True, 5.5]

## Indexed
Each item in a list has an index.
The first item is at index 0.

Example:
fruits[0] gives the first item

## Can grow or shrink
Lists are flexible.
You can add new items or remove old items anytime.

Example:
fruits.append("grapes")
fruits.remove("apple")

## Why use lists?
Lists are useful when you want to store many related values in one variable.

Examples:
- student names
- marks
- shopping items
- daily tasks

## Important points
- Lists use square brackets `[ ]`
- They are ordered
- They are mutable
- They allow duplicates
- They can store different types
